Setup projects dapat digunakan untuk membangun berbagai paket penerapan. 
Setup projects digunakan untuk membuat file Penginstal Windows (.msi), yang digunakan untuk mendistribusikan 
aplikasi kita untuk penginstalan di komputer lain atau web server.
