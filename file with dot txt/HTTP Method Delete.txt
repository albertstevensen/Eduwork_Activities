Tugas:
Menurut anda, assertion untuk Method DELETE.
Selain hanya status code apakah ada lagi? Coba jelaskan.

Jawaban:
Berdasarkan hasil pencarian saya assertion untuk Method DELETE, 
hampir semua automation test khususnya di cypress menggunakan assertion untuk Method DELETE
dengan status code. Status code sendiri adalah jawaban server website atas permintaan data oleh browser kita
kesimpulan yang saya dapat adalah assertion untuk Method DELETE dengan status code hanya untuk melakukan validasi apakah
data pada API sudah terhapus atau belum, karena data yang dihapus berdasarkan ID otomatis seluruh data dengan ID tertentu akan dihapus.
Jadi status code akan memberi respon ke kita bahwa seluruh data yang kita request untuk dihapus sudah terhapus. 
Sehingga hanya response dari status code yang dapat di validasi, tidak dapat menggunakan response yang lain.
